Welcome to my RESTful API service!

The purpose of this API is to allow users to understand how demographics play a role in which vaccine type (Moderna, Pfizer and/or Janssen) an individual recieves.

This API allows users to:

  • access COVID-19 vaccine data from the Centers for Disease Control (CDC)
  • report statistics on vaccine distribution.
  • retrieve a personalized reccomendation based on demographics. (in development)

Also if you are viewing the API's web interface (this page), feel free to click on the "View Data" tab to check out some cool interactive visuals to get an overview of the data!

If you plan on accessing the API's endpoints through your own frontend application, check out the interactive docs from Swagger UI to get an understanding of the data this API can provide.

All initial data is provided by the Centers for Disease Control, a division of the United States Department of Health and Human Resources (HHS). To learn more about the HHS and their mission please visit their website.
